sqr

A fast, keyboard-first TUI for exploring SQLite databases

GitHub Crates.io

Install

cargo install sqr

Or build from source:

git clone https://github.com/Bunchhieng/sqr.git
cd sqr && cargo build --release

Usage

sqr database.db
sqr database.db --read-write
sqr database.db --page-size 50

Export:

sqr export --db database.db --table users --format csv --out users.csv
sqr export --db database.db --query "SELECT * FROM users" --format json --out users.json

Features

Fast startup
Keyboard-first
Three-pane layout
Table exploration
ER Diagram visualization
Row editing inline
SQL scratch pad
Export CSV/JSON

Keybindings

  • Tab Panes
  • ↑/↓ Navigate
  • ←/→ Pages
  • Enter Execute/Edit
  • s Cycle views
  • d ER Diagram
  • e SQL Editor
  • Ctrl+E Full editor
  • Shift+Enter Newline
  • Ctrl+C Clear
  • Ctrl+U/K Clear line
  • Ctrl+A/E Start/End
  • Ctrl+W Del word
  • / Filter
  • ? Help
  • q Quit

View Modes

Press s to cycle:

Rows

Browse table data with pagination

Schema

View table structure and constraints

Diagram

Visual ER diagram with relationships
